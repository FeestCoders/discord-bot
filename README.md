## 👾 Installation

Use [nodejs](https://nodejs.org/en/download/) to install the dependencies.

```bash
npm i
```

add a .env file
```
TOKEN= your discord bot token
PREFIX=!!
```

## 🔭 Usage

```javascript
npm start
```
